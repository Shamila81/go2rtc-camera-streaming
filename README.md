\## Project Overview



Implementation of a reliable RTSP-based camera streaming pipeline using Go2RTC, integrated with Home Assistant and containerized with Docker.



\## Project Goals



\- Set up Go2RTC as a standalone system

\- Integrate Go2RTC with Home Assistant

\- Configure Docker containerization

\- Enable real-time camera streaming through RTSP

\- Test performance and reliability
